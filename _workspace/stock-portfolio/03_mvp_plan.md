# MVP 계획

## MVP 사용자

초기 사용자는 Numbers로 포트폴리오를 관리하던 소유자 1명이다. 다만 웹사이트로 확장할 수 있게 여러 사용자가 자기 포트폴리오를 입력하는 구조를 막지 않는다. MVP 예시 데이터는 비식별 샘플로 넣을 수 있다.

## 첫 화면

첫 화면은 대시보드다.

- 총 평가금액
- 총 매입금액
- 평가손익
- 수익률
- QQQ/S&P500/국내주식 비중
- 투자자별 비중
- 최근 월/일 성과 차트
- 마지막 데이터 업데이트 시각
- 가격/환율 업데이트 상태

## 핵심 화면

### Holdings

- 보유 종목 테이블
- 투자자, 계좌, 자산군, 통화 필터
- 수량/현재가/평단가 수동 수정
- 평가금액, 손익, 수익률 자동 계산

### Allocation

- QQQ, S&P500, 국내주식 비중
- 연금/IRP/직투 비중
- 투자자별/계좌별 분해

### Performance

- 월별 성과
- 일별 성과
- 입금/출금 조정 후 투자 수익금
- 누적 수익과 월/일 수익

### Import

- XLSX 업로드
- Numbers에서 내보낸 시트 이름 감지
- 매칭 결과 미리보기
- 저장 전 계산 검증

### Price Updates

- Yahoo Finance chart endpoint로 미국 주식/ETF 현재가 가져오기
- Yahoo Finance chart endpoint의 `KRW=X`로 USD/KRW 환율 가져오기
- 마지막 성공 시각과 실패 상태 표시
- quote 5분, fx 1시간 캐시 정책 반영

## 우선순위

1. 비식별 샘플 데이터 기반 대시보드
2. 보유 현황 테이블과 계산식 재현
3. Yahoo Finance 기반 미국 주식/환율 fetch, 캐시, 실패 상태 표시
4. 월별/일별 성과 스냅샷 모델
5. XLSX import 미리보기

## 제외 범위

- 매수/매도 추천
- 자동 주문 또는 브로커 연동
- 실시간 시세 보장. MVP 가격은 지연/캐시된 참고 가격으로 표시한다.
- 세금 계산 자동화
- 공개 공유 페이지
- 한국 주식 자동 가격 업데이트
- 서버 저장, 회원 인증, 팀 공유

## 제품 원칙

- Numbers에서 하던 계산을 웹에서 더 구조적으로 재현한다.
- 금융 조언처럼 보이는 문구를 피한다.
- 원본 파일과 민감 데이터의 저장 위치를 명확히 한다.
- 현재가와 환율은 항상 기준 시각을 함께 보여준다.
- 가격 데이터가 실패하거나 지연되면 마지막 성공값과 오류 상태를 분리해서 보여준다.
